# MK Jewels Assistant

The MK Jewels Assistant is an automated audio capture, processing, and triage system designed to monitor real-time microphone input, transcribe interactions via Gemini speech-to-text, and run intelligent triage using Gemini and OpenRouter models. The system logs these pipeline sessions into a local database, fires real-time console alerts based on configurable thresholds, generates structured daily activity reports, and serves a modern web-based monitoring dashboard.
